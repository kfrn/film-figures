### Local setup

Dependencies:
* [Elm](https://guide.elm-lang.org/install.html)
* [Node](https://nodejs.org/en/download/)

```
git clone git@github.com:kfrn/film-figures.git
cd film-figures/
elm-package install
npm install -g create-elm-app
elm-app start
```

<!-- To run tests:
```
elm-app test
